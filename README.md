# Rubrica_3_Mineria-de-Datos

- Yairineth Camargo
- Mariana Cala
- Edgardo Pacheco

This project was developed as a final Data Mining assignment focused on the classification of Iris flower species using Machine Learning techniques.

The objective is to predict the species of a flower based on four morphological measurements:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

Several classification models were trained and evaluated, including:
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Decision Tree
- Random Forest

The project also includes an interactive dashboard built with Streamlit for data visualization and real-time predictions.
